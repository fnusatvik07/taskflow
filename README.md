# TaskFlow

A tiny task manager. This is the live demo project for the **Claude Code for Builders**
class (Data Sense · Launchpad).

> This README is intentionally thin. One of the class demos is asking Claude Code to
> flesh it out, so leave room for it.

## Run it

```
pip install -r requirements.txt
python app.py
```

Then open http://localhost:5000

## What's inside (and the demo each part feeds)

- `app.py` is the main entry point
- `taskflow/registration.py` has an empty-form bug to fix
- `taskflow/calculator.py` has pure functions to write tests for
- `taskflow/auth.py` has callback-style auth to refactor
- `data/tasks.csv` is sample data to analyze
